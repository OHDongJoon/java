-inline-
 : TXT처럼 옆으로 쌓는 특징

  1) 한 줄에 여러 요소를 포함할 수 있음 = 기본 너비값: 컨텐츠 너비값

  2) 너비값, 높이값을 가질 수 없음. (폼 엘리먼트, 이미지 제외)

  3) 상, 하단 마진을 가질 수 없음.

  4) block은 inline을 포괄하는 더 큰 개념( inline 태그 안에 block 태그 사용 금지 )
